El usuario pide una ruta a la aplicacion y esta se comunica con el pod del usuario para obtener la información que es renderizada y mostrada al usuario.
Por otro lado para que la aplicación pueda mostrar el mapa hace uso de Leaflet, le pide el fragmento que quiere mostrar y este le devuelve las imagenes para poder mostrar el mapa(Esto se realiza automaticamente gracias a la libreria de Leaflet)
El usuario también puede compartir rutas con amigos, para ello la aplicación generará una notificación en el pod del amigo.
